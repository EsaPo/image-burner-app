Create bootable USB drives for Linux and Windows

A modern, user-friendly application for creating bootable USB drives on Linux systems. Supports both Linux distributions and Windows installation media.

Features:

✓ Linux ISO support (Ubuntu, Fedora, Debian, etc.)
✓ Windows 10/11 bootable USB (WoeUSB-style)
✓ UEFI:NTFS bootloader for maximum compatibility
✓ Compressed .img.gz file support (auto-decompress)
✓ Nero (.nrg) file support with auto-conversion
✓ SHA256 checksum verification
✓ Safe Mode & Advanced Mode
✓ Real-time progress tracking
✓ Proper partition table refresh (fixes Nemo/Nautilus)

Platform Requirements:

This application is designed exclusively for Linux and requires:
• parted, dd, mkfs.ntfs, mkfs.vfat
• rsync, wipefs, lsblk
• wget or curl (for UEFI:NTFS download)
• PolicyKit (pkexec) for admin rights
• GRUB or syslinux (optional) for Legacy BIOS support
• nrg2iso (optional) for Nero file conversion

Why Linux-only?

While Rufus is excellent for Windows, Linux lacks a good tool for creating Windows installation USB drives. This app fills that gap with the same proven UEFI:NTFS method!
