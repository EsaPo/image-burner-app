This is an app which you can create bootable USB drives for Linux and Windows. App supports both Linux distributions and Windows 10/11 installation media with Rufus-style UEFI boot.

ToDo:

```
# First clone this repo
git clone git@github-esapo:EsaPo/image-burner-app.git

# Install dependencies
npm install

# Run/test app with command
npm start

# To build distributable packages (AppImage, deb, tar.gz) run command
npm run build:linux
```
